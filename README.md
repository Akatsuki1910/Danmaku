## 個人用メモ（nishionyama）

fork したレポジトリが最新版に追いつかないなら

```git
git remote add upsatream https://github.com/[parent's repositoryname].git
git fetch upstream
git merge upstream/master
```

で最新版に追いつくことができる。
remote した upstream 枝は

```git
git remote remove upstream
```

で消しておくと吉。
